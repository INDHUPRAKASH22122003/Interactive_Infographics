												Interactive Infographic

					This project is an interactive infographic that visualizes data in a dynamic and engaging manner using HTML, CSS, JavaScript, and Chart.js library.

## How to Run:

To run the image slider, follow these steps:

1. Download/Clone Repository: Download or clone the repository to your local machine.

2. Open HTML File: Navigate to the folder containing the `mainpage.html` file in your file explorer.

3. Open the HTML File:
     - Open `mainpage.html` in a web browser.
     - Alternatively, if you have a local server set up:
     - Launch the server.
     - Access the project via the provided local server address.

## Usage:

- Data Input:
  - Enter data into the provided input field.
  - Separate each data point by commas.
  - Click on the "Generate Chart" button to visualize the data.

- Chart Interaction:
  - Hover over the bars to view tooltips with additional information about each data point.
  - The chart animation showcases data transitions when new data is provided.

## Technologies Used:

- HTML
- CSS
- JavaScript
- Chart.js Library

## Additional Notes:

- Ensure an active internet connection to fetch Chart.js library if not locally available.
- For optimal viewing experience, use a modern web browser.
